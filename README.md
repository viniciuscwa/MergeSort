# MergeSort
Projeto de S.O com uso de Multithreading

## Como compilar :
```
gcc -pthread main.c -o threads  
```
## Como executar :
```
./threads 4 arq1.txt arq2.txt arq3.txt arq4.txt arq5.txt
```
## Codigo:
- main.c

## Executavel:
- threads
