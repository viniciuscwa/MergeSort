# MergeSort
Projeto de S.O com uso de Multithreading e Merge Sort.

## Como compilar :
```
gcc -pthread main.c -o threads  
```
## Como executar :
```
./threads 4 arq1.txt arq2.txt arq3.txt arq4.txt arq5.txt > output.txt
```
4 = numero de threads

- main.c

## Executavel:
- threads
